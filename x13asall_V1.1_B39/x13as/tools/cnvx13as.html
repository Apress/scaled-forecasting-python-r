CNVX13AS - an X-13ARIMA-SEATS spec file conversion utility

Purpose:

To convert input specification files from Version 0.3 of X-12-ARIMA to a format compatable with X-13ARIMA-SEATS.

This utility is a DOS utility - the examples below assume you are running the utility within a DOS command window, and are running the utility in the same directory that the utility is stored in.

Syntax:

cnvx13as oldspec newspec

where:

oldspec is the filename of an input specification file for Version 0.3 of X-12-ARIMA;

newspec is the filename of a input specification file for Version 1.0 of X-13ARIMA-SEATS that was formed from the specs and arguments found in oldspec.

Note:

CNVX13AS assumes that the input is a valid input specification file for Version 0.3 of X-12-ARIMA; no attempt is made to verify if the contents of the file are in the correct syntax.

Comments written in the old spec file are transferred to the new, though not in the same position in the file.

Spec names and spec arguments from the old input file are converted to lower case in the translated input file.

This program was written with the Icon programming language, with special routines provided by the Icon program library. For more information on Icon, see "The Icon Programming Language, Third Edition" by Ralph E. Griswold and Madge T. Griswold, published by Peer-to-Peer Communications.
